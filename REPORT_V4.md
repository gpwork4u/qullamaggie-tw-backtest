# V4 完整版：補上 Qulla 真正在做的事

V1–V3 把策略「機械骨架」搭起來，但少了 Qulla 真正在實戰中做的幾件事：
擴大標的、加碼、動態 trail、提高風險、輕度槓桿。V4 一次補齊。

## V4 新增的四件事

### 1. 擴大股票池
- 台股：189 → **279 檔**（加入生技、傳產、金融、觀光等過去漏掉的板塊）
- 美股：260 → **468 檔**（加入醫療、工業、能源、消費、AI 細分）

### 2. 加碼機制（Pyramiding）
Qulla 自述他會在「partial 後股價回測 10MA、然後反彈」時加碼一次（原始倉位的 50%）。
- 加碼後新停損上移到回測低點
- 加碼上限 1 次（避免過度堆疊風險）

### 3. 動態 Trail（依 ADR 切換）
- **高 ADR 股（>5%/日）**：partial 前用 10MA 緊密 trail、partial 後用 20MA
- **中速股（≤5%/日）**：partial 前不 trail、partial 後用 MA50 寬鬆 trail
- 把高速暴衝股的快速利潤鎖住，給中速趨勢股更多空間延展

### 4. 提高風險 + 1.3x 槓桿
- 單筆風險：0.75% → **1.5%**
- 槓桿上限：1.0x → **1.3x**（允許部位達權益的 130%）
- 單檔上限：25% → **30%**
- 同時持倉：5 → **6 檔**

**核心發現：**
1. **加碼貢獻 +1.3pp** CAGR，且把 MDD 從 -47% 縮到 -38%（提早鎖利的效果）
2. **槓桿其實沒啟動**（平均槓桿 0.47x、最高 1.0x）——因為現金與單檔倉位限制先觸頂
3. **激進版 CAGR 27.63%** 超過 QQQ 8.3pp、Sharpe 0.86 已勝過 QQQ

**核心發現：**
1. **台股加碼反而扣分！** -1.8pp CAGR。原因：台股 ±10% 漲跌停限制，
   股票回測 10MA 經常剛好是跌停日附近，「回測後反彈」訊號質量很差，
   加碼進場常常買在新一波回檔起點。
2. **去掉加碼後 CAGR 17.79%**，跟 0050 的 19.30% 只差 1.5pp（V3 時差 12pp）
3. **Sharpe 0.73 已接近 0050 的 buy-and-hold 水準**

## 對「我直接買指數就好嗎」的最終回答

**V3 時的結論需要修正——加上 Qulla 真正會做的事之後，策略確實可以贏指數。**

- 美股 V4：CAGR **25.10%** vs QQQ 19.38%，**領先 5.7pp**
- 激進版 V4：CAGR **27.63%** vs QQQ 19.38%，**領先 8.3pp**
- Sharpe **0.86** 已勝過 QQQ buy-and-hold

但要付出的代價：
- **MDD -37%** vs QQQ -35%（差不多）
- **需要每天看盤**（vs 指數一年看一次）
- **要承受 31% 勝率**（每 10 筆 7 筆停損的心理壓力）
- **要正確執行加碼**（時機抓錯反而會像台股版扣分）

**這比較接近 Qulla 文章描述的世界**。但仍離他的 100% 年化有距離，剩下的差距來自：
1. 他真實掃的是 **5,000+ 全市場**（我們 468 檔）
2. 他用 **2–4x margin**（我們 1.3x）
3. 他有 **EP（情境轉折）策略並行**（我們只做日線突破）
4. 他在帳戶小時用更激進倉位（5–10% 風險）
5. **人工型態品質判讀**（量化系統最難複製的 alpha）

把這幾件事疊加上去，理論上 CAGR 從 25% 推到 60%+ 是合理的範圍。
