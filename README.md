## 核心結論

1. **策略「指紋」完全對得上 Qulla 原版**：勝率 18–26%、賠率 3–7 倍、利潤集中在極少數巨贏家。
2. **大盤過濾 + 冷卻期顯著降低回撤**：V3 把最大回撤從 -47% 縮到 -34%，2024 損失從 -25% 縮到 -10%。
3. **CAGR 仍輸 0050**：台積電 50% 權重 + AI 大行情，是這套策略在台股的天花板。
4. **這套策略真正的價值不是「贏過 0050」，而是「在 0050 大跌時你還在賺」**：2018、2022 熊市顯著佔優。

## 淬煉後的策略規則

完整規則見 [`STRATEGY.md`](STRATEGY.md)。核心 5 條：

1. **強者恆強**：過去 60 日 ≥ +30% 且 120 日 ≥ +20%，距 60 日高 ≤ 15%
2. **整理才買**：收盤 > 50MA > 100MA
3. **突破才動**：收盤破 15 日高 + 量能 ≥ 1.5 倍均量 + 非漲停
4. **嚴格停損**：停損 = 進場當日 Low，距離 ≤ 1.5×ATR，超過就放棄
5. **分段了結**：1R 賣 1/3 + 移至 BE + MA50 收盤跌破出剩餘

V3 額外加入：
- 大盤過濾：0050 < 200MA 時停止開新倉
- 弱大盤減倉：0050 < 50MA 時單筆風險砍半
- 冷卻期：同檔股票 20 個交易日內已停損過則跳過

## 快速開始

```bash
# 1. 安裝依賴
pip install yfinance pandas numpy

# 2. 下載資料（首次約需 5–10 分鐘）
python download_data.py

# 3. 跑回測
python backtest.py       # V1：基線
python backtest_v2.py    # V2：放寬 trail
python backtest_v3.py    # V3：加入大盤過濾（推薦）

# 4. 分析 2024 漏單
python analyze_2024.py
```
