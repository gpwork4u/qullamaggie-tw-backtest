# Qullamaggie 台股回測報告

## 一、淬煉的核心規則（見 `STRATEGY.md`）

把原版 11 條心法壓縮成 5 條可量化、可回測的規則：

| # | 規則 | 量化定義 |
|---|---|---|
| 1 | 強者恆強 | 過去 60 日 ≥ +30% 且 120 日 ≥ +20%，距 60 日高 ≤ 15% |
| 2 | 整理才買 | 收盤 > 50MA > 100MA |
| 3 | 突破才動 | 收盤破 15 日高 + 量能 ≥ 1.5 倍均量 + 非漲停 |
| 4 | 嚴格停損 | 停損 = 進場當日 Low，距離 ≤ 1.5×ATR，超過就放棄 |
| 5 | 分段了結 | 1R 賣 1/3 + 移至 BE + MA50 收盤跌破出剩餘 |

## 二、回測設定
- 期間：2018-01-02 ~ 2025-12-30 (約 8 年)
- 股票池：189 檔台股（大型權值 + 中型 100 + 高動能股）
- 初始：100 萬台幣
- 成本：手續費 0.1425%、證交稅 0.3%、滑價 0.2%
- 每筆風險：0.75% 帳戶資金
- 同時持倉：最多 5 檔
- 基準：0050.TW 買入持有

## 四、最關鍵的發現

### 1. 策略「指紋」完全對上 Qulla 原版
- 勝率 20–26% ✓（Qulla 自述 25–30%）
- 賠率 3–5 倍 ✓（Qulla 強調「讓贏的時候足夠大」）
- 大部分利潤來自極少數巨贏家（V2 前 5 筆貢獻整體一半以上獲利）

### 2. V1 vs V2 的差距全部來自「移動停利」一個參數
- V1 用 MA20 trail，10 日就啟動 → 把長榮 (2615) 在 90% 就洗掉
- V2 改 partial 後才啟動、改 MA50 → 同樣的 2609 抱到 +217%

這驗證了 Qulla 那句話：**「贏的時候足夠大」比「進場準」重要十倍。**

### 3. 策略仍輸 0050——為什麼？
| 年份 | V2 策略 | 0050 | 差距 |
|---|---|---|---|
| 2018 | +17% | -5% | **+22%** |
| 2019 | -5% | +36% | -41% |
| 2020 | **+54%** | +30% | +24% |
| 2021 | +11% | +20% | -9% |
| 2022 | -13% | -22% | **+9%** |
| 2023 | +25% | +27% | -2% |
| 2024 | -25% | +49% | **-74%** |
| 2025 | +27% | +37% | -10% |

**致命傷在 2024 年：策略 -25%、0050 +49%、差距 74 個百分點。**

原因：2024 年 AI 行情是台積電 (2330) 帶動的「大型股漫長爬升」，不是 Qulla 偏愛的「中小型暴衝突破」。台積電一年漲 80%，但全年從沒出現「整理 2-8 週後 1.5×ATR 內的乾淨突破」，所以策略幾乎沒參與到。0050 因為權重 50% 在台積電，吃滿這波。

### 4. 策略在熊市顯著贏過 0050
- 2018：+22%
- 2022：+9%
- 顯示「強勢股突破 + 嚴格停損」在大盤回檔時保護力強，回撤先於指數結束。

## 五、給實戰者的三個結論

1. **Qullamaggie 系統在台股可運作，但威力遠不如美股**
   - 美股強勢股一年動輒 +500%~1000%（NVDA、SMCI），台股最強的 2609 一年 +200% 已是頂級
   - 賠率天花板較低 → 同樣的勝率，期望值較低

2. **真正能壓制 0050 的，是「不要被洗下車」**
   - 整套策略的命脈不是進場條件，是 trail 寬度
   - 若你執行時 MA20 一破就跑，等於把策略的賠率優勢親手廢掉

3. **此策略不適合「全部資金 all-in」**
   - 最大回撤 47% 是正常範圍，不是 bug
   - 適合作為「衛星部位」（10–30% 資金），核心仍應持有 0050 或 006208

## 六、可進一步優化的方向
- 加入「大盤過濾」：0050 跌破 200MA 時減半倉位（避免 2024-style 系統性錯過）
- 動態切換：大型股趨勢期用 MA50 trail、中小型暴衝用 10MA trail
- 加入「補倉」規則：原版有突破後在均線回測時加碼一次
- 擴大股票池：目前 189 檔仍偏向中大型，缺少興櫃轉上市的爆發股
