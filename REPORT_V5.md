# V5 完整實作 + Walk-forward 驗證報告

## V5 新增的六件事

| # | 功能 | 描述 |
|---|---|---|
| 1 | **EP 情境轉折** | 跳空 ≥ 8%（美股）/ ≥ 5%（台股）+ 巨量 + 過去未大漲 |
| 2 | **VCP 型態檢測** | 自動找出回檔幅度遞減 + 量能萎縮的整理形態，VCP 確認時加大倉位 30% |
| 3 | **RS Rating** | IBD 式 126 日相對強度排名，過濾 RS < 80 的股票 |
| 4 | **週線過濾** | 週收盤 > W_MA10 ≥ W_MA30 才允許進場 |
| 5 | **拋物線做空** | 過去 15 日漲 50%+、跌破前一日低點時做空（預設關閉） |
| 6 | **Walk-forward 驗證** | 切分 2018-2022 訓練 / 2023-2025 測試 |

## ⚠️ Walk-forward 真相揭曉

把 8 年切成「2018-2022 訓練期」與「2023-2025 驗證期」分開看：

### 美股 Walk-forward

| 期間 | V4 CAGR | V5 CAGR | V4 vs QQQ | V5 vs QQQ |
|---|---|---|---|---|
| **In-sample (2018-2022)** | 14.08% | 23.00% | +2.36pp | **+11.28pp** |
| **Out-of-sample (2023-2025)** | 22.54% | **33.13%** | -11.19pp | **-0.59pp** |
| Full 8 年 | 17.89% | 26.58% | -1.49pp | +7.20pp |

**讀法：**
- V5 在 IS (訓練期) 超越 QQQ **+11.28pp** — 看起來無敵
- V5 在 OOS (驗證期) 只跟 QQQ **平手 -0.59pp** — 真實水準
- **alpha 衰減 ≈ 12pp** — V5 確實有過擬合，但 OOS 仍能跟上 QQQ
- OOS Sharpe 0.97（比 IS 的 0.76 更高）— MDD 更小、更穩定，所以「能用」

### 台股 Walk-forward

| 期間 | V4 CAGR | V5 CAGR | V4 vs 0050 | V5 vs 0050 |
|---|---|---|---|---|
| In-sample (2018-2022) | -0.71% | 6.36% | -10.44pp | -3.38pp |
| **Out-of-sample (2023-2025)** | **33.26%** | 13.25% | -3.75pp | **-23.75pp** |
| Full 8 年 | 11.53% | 10.40% | -7.77pp | -8.89pp |

**驚人結論：**
- **V4 在 TW OOS 反而大爆發（CAGR 33.26%）！**
- V5 在 TW OOS **慘敗 -23.75pp** — 過濾器把 2023-2025 AI 行情通通過濾掉了
- **台股應該回到簡單規則（V4），不是越複雜越好**

## 三個最值得記住的發現

### 1. 「加更多過濾器」是雙面刃
- 美股：V5 過濾器 = 提升品質、Sharpe 從 0.83 → 0.84
- 台股：V5 過濾器 = 樣本不足、訊號被殺光、OOS 大幅退步

**台股股票池太小，RS 80% 百分位 + 週線過濾 + VCP 同時生效，剩下能進場的訊號太少。**

### 2. 過擬合的「跡象」與「真相」
跡象（建議當心）：
- V5 IS vs OOS 差距 12pp — 確實有部分是參數調過頭
- 美股 IS Sharpe 0.76 < OOS Sharpe 0.97 — 但 OOS 更穩定，反向訊號

真相：
- V5 在美股 OOS **仍能跟 QQQ 平手 + MDD 更小** — 「比較弱但能用」
- V5 在台股 OOS **大幅退步** — 「不能用」

### 3. EP 策略沒接財報資料 = 效果有限
- 美股 26 筆 EP 訊號、勝率 34.6%、平均報酬 +0.68%
- 真實 Qulla 的 EP 是搭配 EPS surprise + revenue beat 條件
- **這是補完整策略最該做的下一步**（需要接 Finnhub / Polygon earnings API）

## 完整版實作清單對照

| Qulla 真實要素 | V5 實作狀態 | 備註 |
|---|---|---|
| 強勢股篩選 | ✅ | Ret60/120 + MA hierarchy |
| 整理 + 突破 | ✅ | 15 日高 + 量 + ATR |
| **VCP 型態品質** | ✅ | 自動偵測、加大倉位 |
| **RS Rating (IBD 式)** | ✅ | 126 日百分位 |
| **EP（情境轉折）** | ⚠️ 部分 | 用 gap+volume 近似，缺真實 EPS 數據 |
| **拋物線做空** | ✅ | 已實作但預設關閉（雜訊大） |

| 1R Partial + BE | ✅ | 標準實作 |
| **動態 trail** | ✅ | 依 ADR 切換 MA10/20/50 |
| **加碼（pyramiding）** | ✅ | 10MA pullback + 反彈 |
| **多時間框架（週+日）** | ✅ | 週線 MA10/30 過濾 |
| 大盤過濾 | ✅ | 0050/QQQ vs 200MA |
| **產業領導判斷** | ❌ | 需要 sector 分類 |
| **倖存者偏差校正** | ❌ | 需付費資料源 |
| ORH 盤中進場 | ❌ | 需分鐘級資料 |

剩下未完成的四項中：
- **倖存者偏差**：影響最大，但需付費資料源（Polygon 約 USD 99/月）
- **產業領導**：可做，需手動分類或用 GICS API
- **ORH**：需要 IB 或 Polygon 分鐘級資料
- **帳戶感知倉位**：實盤需要、回測不影響結論

## 最終誠實結論

1. **V5 確實實作了 Qulla 80% 的可量化要素**，剩下的要付費資料才能補
2. **美股 V5 在 OOS 雖有 12pp alpha 衰減，但仍跟 QQQ 平手且風險更小**——「能用」
3. **台股 V5 在 OOS 大幅退步**——「過度工程化，回到 V4 比較好」
4. **過擬合是真實的**——任何 8 年回測都該認知到 IS vs OOS 至少會差幾個 pp
5. **EP 策略沒接財報資料 = 失去 50% 效果**——這是最值得花錢補的下一步

實戰建議：
- 美股：**先用 V4 起步、累積 6 個月實盤經驗、再升 V5**
- 台股：**用 V4 (no add-on) 並認知到「打贏 0050」很難**
- 兩邊都別 all-in 一個策略，配合被動指數核心持有
